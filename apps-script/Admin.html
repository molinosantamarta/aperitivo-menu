Come funziona Menumal

Cosa gestisce cosa

  • Menumal e il pannello admin per prodotti, prezzi, descrizioni, visibilita e disponibilita.
  • Google Sheet e il database operativo dietro Menumal.
  • Agri-Menu e il menu pubblico su GitHub Pages aperto dai clienti via QR.

Flusso dati

  • Tu modifichi un prodotto da Menumal.
  • Apps Script salva i dati nel Google Sheet.
  • Agri-Menu legge quei dati e aggiorna il menu mostrato ai clienti.

Quando usare Menumal

  • Aggiungere un prodotto semplice.
  • Cambiare prezzo, titolo o descrizione.
  • Mettere visibile, nascosto, in arrivo o esaurito.

Quando chiedere modifiche al codice

  • Card legacy personalizzate.
  • Carousel, animazioni, layout e stile.
  • Cambi strutturali di frontend, CSS o logica JS.

Deploy

  • Le modifiche ai contenuti fatte in Menumal non richiedono deploy GitHub.
  • Le modifiche grafiche o strutturali del menu pubblico richiedono aggiornamento del codice e pubblicazione.
